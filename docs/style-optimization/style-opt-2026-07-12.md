## 具体修改内容

### 1. global.css - 补充缺失语义变量

前两日建立了 `--color-danger-soft` / `--color-warn-soft` 等语义色别名，但搜索发现大量页面仍硬编码以下模式：
- `rgba(43, 108, 255, 0.2)` — 主色边框
- `rgba(122, 76, 255, 0.06~0.12)` — 辅色柔和背景
- `rgba(26, 170, 85, 0.25)` — 成功色边框
- `rgba(26, 170, 85, 0.15)` — 成功色焦点环

新增 4 个语义变量作为"中间层"：

```css
--color-primary-border: rgba(43, 108, 255, 0.2);
--color-accent-soft: rgba(122, 76, 255, 0.08);
--color-success-border: rgba(26, 170, 85, 0.25);
--color-success-ring: rgba(26, 170, 85, 0.15);
```

暗色模式同步覆盖：

```css
--color-primary-border: rgba(91, 140, 255, 0.3);
--color-accent-soft: rgba(122, 76, 255, 0.14);
--color-success-border: rgba(26, 170, 85, 0.35);
--color-success-ring: rgba(26, 170, 85, 0.22);
```

**设计意图**：消除 `rgba(43, 108, 255, 0.2)` 等边框色在各页面反复硬编码的问题，后续调整配色只需改一处。

### 9. PasswordTool.tsx - 强度等级色值

密码强度计的"极弱"和"弱"等级此前使用硬编码十六进制值，与 `--color-error` / `--color-warning` 完全一致，替换为 CSS 变量。

| 等级 | 优化前 | 优化后 |
|------|--------|--------|
| 极弱 | `'#e5484d'` | `'var(--color-error)'` |
| 弱 | `'#f59e0b'` | `'var(--color-warning)'` |

注："一般"(`#eab308`)、"强"(`#22c55e`)、"很强"(`#15803d`)三级保留原值，因为它们是强度渐变中的中间色阶，不属于语义色体系。

### 10. TextSimilarityTool.tsx - 修复错误 fallback

`similarityLevel` 函数使用 `var(--color-xxx, fallback)` 形式，但 fallback 值与变量不匹配（如 `--color-success` 的 fallback 写成 `#16a34a` 实为 `--color-insert` 的值），属于潜在 bug。移除所有 fallback，简化为纯变量引用。

| 等级 | 优化前 | 优化后 |
|------|--------|--------|
| 高度相似 | `'var(--color-success, #16a34a)'` | `'var(--color-success)'` |
| 较为相似 | `'var(--color-primary, #2b6cff)'` | `'var(--color-primary)'` |
| 部分相似 | `'var(--color-warning, #d97706)'` | `'var(--color-warning)'` |
| 差异较大 | `'var(--color-danger, #dc2626)'` | `'var(--color-danger)'` |

## 保留未修改的场景

以下硬编码值经评估后保留原样，不属于样式不一致：

1. **PasswordTool 强度渐变中间色阶**：`#eab308`（一般）、`#22c55e`（强）、`#15803d`（很强）是强度计五级渐变中的中间色，不属于语义色体系
2. **diff.astro 字符级高亮**：`rgba(229, 72, 77, 0.35)` 等使用 0.35 不透明度（高于 `--color-danger-border` 的 0.3），用于字符级 diff 高亮的特殊可见性需求
3. **MimeTool 类别颜色**：9 个 MIME 类别各有独立颜色（文档蓝、图片绿、音频橙等），属于分类色板而非语义色
4. **MarkdownTool 导出模板**：`#f5f5f5`、`#1f2937` 等用于导出 HTML 的内联样式，需自包含不能引用站点 CSS 变量
5. **json.astro 对象括号色**：`#c9366f`（对象括号）和 `#f0709a`（暗色模式）是语法高亮中的装饰色，无对应语义变量
