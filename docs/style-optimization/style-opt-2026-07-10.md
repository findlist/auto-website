## 具体修改内容

### 1. global.css - 设计令牌扩展

新增以下设计令牌，统一全站引用：

- `--color-border-strong`：更强边框色（hover 态）
- `--color-primary-soft` / `--color-primary-ring`：主色柔和背景与焦点环
- `--color-success-soft` / `--color-warning-soft` / `--color-error-soft`：语义色柔和背景
- `--color-accent`：辅助强调色（紫色 #7a4cff）
- `--shadow-lg`：大阴影层级
- `--ease-out` / `--transition-fast` / `--transition-base` / `--transition-slow`：统一过渡曲线
- `--z-header` / `--z-overlay` / `--z-toast`：z-index 层级规范

暗色模式同步新增对应变量。

### 2. global.css - 通用样式增强

- **导航**：高度 56px → 60px，背景使用 `color-mix` 半透明，backdrop-filter blur 8px → 12px，品牌图标增加 box-shadow 与 hover 缩放旋转动效，导航链接增加 font-weight: 500
- **导航激活态**：新增 `aria-current="page"` 支持，激活时显示主色文字 + 柔和背景
- **按钮**：新增 `:active` 缩放反馈，`--primary` 增加 box-shadow 光晕，过渡属性细化
- **页脚**：间距 32px → 36px，margin-top 64px → 72px
- **Hero**：间距 48px → 52px，段落行高增加
- **选中文本**：新增 `::selection` 主色高亮
- **滚动条**：自定义 webkit 滚动条样式 + Firefox scrollbar-width
- **页面淡入**：main 区域 `fade-in-up` 动画（0.32s）
- **减少动画**：`prefers-reduced-motion: reduce` 全局禁用动画

### 3. global.css - FAQ 统一样式

将 `.json-faq` 折叠面板样式从 json.astro 迁移至 global.css，**修复了 JWT、Regex、Hash、QR、Password、Color、Diff 等工具页面 FAQ 区域无样式的问题**（这些页面使用 `class="json-faq"` 但未定义对应样式）。

增强项：
- details 圆角 `--radius-sm` → `--radius-md`
- details hover 边框增强
- details[open] 边框主色 + 阴影
- summary hover 背景反馈
- +/− 指示器颜色过渡（展开时变为主色）
- 段落内 code 样式统一

### 4. global.css - 复制成功反馈

新增 `.is-copied` 工具类与 `copy-flash` 动画，复制按钮成功后短暂变绿。

### 5. BaseLayout.astro - 导航激活状态

根据 `Astro.url.pathname` 动态设置 `aria-current="page"`：
- 首页：`/`
- 工具：`/json`、`/base*`、`/hash`、`/jwt*`、`/regex*` 等工具路径
- 博客：`/blog*`
- 关于：`/about`

### 6. index.astro - 首页增强

- **Hero**：双层径向渐变背景（主色 + 辅色），标题 40px → 42px，间距增大
- **工具卡片**：新增 `::before` 左侧主色渐变指示条（hover 时 scaleY 展开），悬停位移 -2px → -3px
- **搜索框**：focus 改为 box-shadow 焦点环（替代 outline），新增 `flex: 1` 自适应宽度
- **分类按钮**：激活态增加 box-shadow 光晕
- **特色卡片**：新增 hover 上浮 + 阴影效果
- **博客卡片**：悬停位移 -2px → -3px
- **标签**：统一使用 `--color-primary-soft` / `--color-primary-ring` 变量
- **过渡**：全部 `0.15s ease` → `var(--transition-base)`（0.18s ease-out）

### 7. json.astro - 工具容器增强

- `.jsontool` 新增 hover 阴影过渡（shadow-sm → shadow-md）
- 移除重复的 `.json-faq` 样式定义（已统一至 global.css）

### 8. blog/index.astro - 博客列表增强

- 博客卡片新增 `::before` 左侧主色渐变指示条（与首页工具卡片视觉风格统一）
- 标签配色统一使用 `--color-primary-soft` / `--color-primary-ring` 变量
- 过渡曲线统一为 `var(--transition-base)`
