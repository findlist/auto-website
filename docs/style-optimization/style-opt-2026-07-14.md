## 具体修改内容

### 1. global.css - 窄屏断点（≤480px）补全

前 4 天的优化覆盖了 768px 平板断点，但 320-480px 手机断点存在细节缺失：导航链接在 320px 屏会换行，底部信息区在窄屏文字拥挤。本次补全。

**新增断点样式：**

| 选择器 | 属性 | 优化前 | 优化后 |
|--------|------|--------|--------|
| `.site-header__inner` (≤480px) | height | 60px | 54px |
| `.site-header__brand` (≤480px) | font-size | 17px | 16px |
| `.site-header__brand-mark` (≤480px) | width / height | 30px / 30px | 28px / 28px |
| `.site-header__nav a` (≤480px) | padding / font-size | 7px 14px / 14px | 6px 10px / 13px |
| `.site-footer__inner` (≤480px) | flex-direction | row | column（居中堆叠） |
| `.container` (≤480px) | padding | 0 20px | 0 16px |

### 2. index.astro - 首页 hero 视觉层次与卡片悬停精细化

**hero 底部主色细线：** 与工具页 `page-hero--tool h1::after` 视觉语言呼应，在 hero 区底部添加 64×3px 主辅渐变细线（opacity 0.7），强化品牌一致性。

```css
.hero::after {
  content: '';
  position: absolute;
  left: 50%;
  bottom: 0;
  transform: translateX(-50%);
  width: 64px;
  height: 3px;
  border-radius: 2px;
  background: linear-gradient(90deg, var(--color-primary), var(--color-accent));
  opacity: 0.7;
}
```

**tool-card / blog-section__card 悬停光晕：** 在原有 `translateY(-3px) + shadow-md` 基础上叠加 `0 0 0 3px var(--color-primary-ring)` 主色光晕，强化"可点击"的可供性提示，与表单元素的 `--focus-ring` 视觉语言一致。

**新增 ≤375px 超小屏断点：**

| 选择器 | 属性 | 优化前 | 优化后 |
|--------|------|--------|--------|
| `.hero__inner h1` (≤375px) | font-size | 30px（继承 768px） | 26px |
| `.hero__lead` (≤375px) | font-size | 15px | 14px |
| `.tools__grid` (≤375px) | grid-template-columns | repeat(auto-fill, minmax(260px, 1fr)) | 1fr（强制单列） |
| `.features__grid` (≤375px) | grid-template-columns | repeat(auto-fit, minmax(280px, 1fr)) | 1fr |
| `.blog-section__grid` (≤375px) | grid-template-columns | repeat(auto-fill, minmax(280px, 1fr)) | 1fr |

### 3. qr.astro - 颜色选择器移动端触控目标

WCAG 2.1 AAA 推荐触控目标最小 44×44px，原 32×32 在移动端偏小。

| 选择器 | 属性 | 优化前 | 优化后 |
|--------|------|--------|--------|
| `.qrtool__color-picker` (≤768px) | width / height | 32px / 32px | 40px / 40px |

### 4. password.astro - 强度条过渡曲线统一

`--transition-base` 在 global.css 定义为 `0.18s cubic-bezier(0.16, 1, 0.3, 1)`，原 `0.25s ease` 与全站过渡节奏不一致。

| 选择器 | 属性 | 优化前 | 优化后 |
|--------|------|--------|--------|
| `.pwtool__strength-fill` | transition | `width 0.25s ease, background 0.25s ease` | `width var(--transition-base), background var(--transition-base)` |

### 5. blog/[...slug].astro - 博客详情页链接可读性

正文链接在加粗字体上下划线显得过厚，且缺少 hover 颜色反馈。

| 选择器 | 属性 | 优化前 | 优化后 |
|--------|------|--------|--------|
| `.blog-article__content a` | text-decoration-thickness | 未设置（浏览器默认） | 1px |
| `.blog-article__content a` | transition | 未设置 | `color var(--transition-fast)` |
| `.blog-article__content a:hover` | color | 未设置 | `var(--color-primary-hover)` |

### 6. blog/[...page].astro - 博客列表卡片悬停反馈统一

与首页 `tool-card` / `blog-section__card` 悬停效果保持一致，叠加主色光晕。

| 选择器 | 属性 | 优化前 | 优化后 |
|--------|------|--------|--------|
| `.blog-card:hover` | box-shadow | `var(--shadow-md)` | `var(--shadow-md), 0 0 0 3px var(--color-primary-ring)` |
