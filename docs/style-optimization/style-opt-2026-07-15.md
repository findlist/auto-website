## 具体修改内容

### 1. global.css - 设计系统源头增强

#### 阴影系统分层重构

将单一 `--shadow-md` 改为分层叠加，模拟自然光照深度，新增 `--shadow-hover` 主色染色阴影用于交互态。

| 变量 | 优化前 | 优化后 |
|------|--------|--------|
| `--shadow-sm` | `0 1px 2px rgba(0,0,0,0.05)` | 分层叠加（保留细节） |
| `--shadow-md` | `0 4px 6px rgba(0,0,0,0.1)` | 分层叠加模拟自然光照 |
| `--shadow-hover` | 未定义 | 新增主色染色阴影 |
| 暗色模式阴影 | 强度偏弱 | 同步增强 |

#### 弹性缓动与过渡时长

| 变量 | 优化前 | 优化后 |
|------|--------|--------|
| `--ease-spring` | 未定义 | `cubic-bezier(0.34, 1.56, 0.64, 1)` 弹性回弹 |
| `--transition-base` | `0.18s` | `0.2s` |
| `--transition-slow` | `0.28s` | `0.32s` |

#### 按钮与交互反馈

- 新增 `min-height: 36px` 保证触控目标
- 悬停时背景填充主色 soft
- 按压过渡加速
- 主色按钮悬停上浮

#### 工具页 hero 增强

```css

  to { transform: scaleX(1); }
```

#### 复制反馈弹性动画

- 使用 `--ease-spring` 弹性缓动
- 新增 `box-shadow: 0 0 0 3px var(--color-success-ring)` 发光效果

#### FAQ 折叠指示条

#### 品牌标志悬停光晕

## 设计原则总结

本次优化遵循以下核心原则：

1. **设计系统源头优先**：所有视觉语言从 `global.css` 的 CSS 变量出发，避免页面级硬编码
2. **分层入场动画**：首屏元素按重要性 0.1s 错开入场，营造叙事感而非同时刷出
3. **方向性反馈**：悬停统一上浮（Y 轴负方向），点击下沉，符合物理直觉
4. **主色染色阴影**：交互态阴影使用主色 ring 变量染色，强化品牌一致性
5. **弹性回弹缓动**：复制成功等正向反馈使用 `--ease-spring` 弹性曲线，增加愉悦感
6. **功能零侵入**：所有修改仅限 CSS，HTML 结构与 JS 行为完全保留
