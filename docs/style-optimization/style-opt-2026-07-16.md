## 具体修改内容

### 1. `src/styles/global.css` - 完善 prose 排版样式

**问题**：原 `.prose` 样式仅覆盖 h1/h2/p/ul/ol/li，缺少 code、pre、a、strong、img、blockquote、table 等元素样式，导致关于页与隐私政策页的 Markdown 内容渲染不一致。

**优化**：

- 补充 `.prose code` 行内代码样式（等宽字体 + 主色文字 + 浅灰背景）
- 补充 `.prose pre` / `.prose pre code` 代码块样式（滚动条 + 圆角边框）
- 补充 `.prose a` 链接样式（主色 + 下划线 + hover 加深）
- 补充 `.prose strong` 加粗样式（增强视觉权重）
- 补充 `.prose img` 图片样式（圆角 + 边框 + 阴影）
- 补充 `.prose blockquote` 引用块样式（左侧主色条 + 柔和背景）
- 补充 `.prose table` 表格样式（边框 + 表头背景）
- `.prose h2` 增加左侧主色渐变细条装饰，增强章节识别
- 补充 `.prose h3` 三级标题样式
- 增加 768px 移动端断点，标题字号收紧
- `.prose h1` 字号从 28px 提升至 32px，增加 `letter-spacing` 与 `line-height`

### 2. `src/styles/global.css` - 优化工具页 hero 大屏表现

**问题**：`.page-hero--tool` 在大屏（≥1024px）下 padding 与字号偏小，视觉冲击力不足。

**优化**：新增 `@media (min-width: 1024px)` 断点：

- padding 从 `44px 0 24px` 提升至 `56px 0 28px`
- h1 字号从 30px 提升至 34px
- p 字号从 15px 提升至 16px

### 3. `src/pages/index.astro` - 首页 hero 响应式字号

**问题**：首页 hero 标题固定 44px，在中等屏幕（如平板）显示偏大，在超宽屏显示偏小。

**优化**：

- `.hero__inner h1` 字号改为 `clamp(30px, 5vw + 16px, 52px)`，小屏 30px、大屏 52px 平滑缩放
- `.hero__lead` 字号改为 `clamp(15px, 1.2vw + 11px, 18px)`，与标题保持视觉节奏
- `max-width` 微调（h1: 720→760，lead: 580→600）

### 4. `src/pages/index.astro` - 工具卡片网格优化

**问题**：`.tools__grid` 使用 `minmax(260px, 1fr)`，在中等屏幕下卡片可能过窄。

**优化**：`minmax(260px, 1fr)` → `minmax(280px, 1fr)`，中等屏幕卡片更宽松。

### 5. `src/pages/index.astro` - 特色图标视觉增强

**问题**：`.feature__icon` 原为纯 emoji 文本，在暗色模式下背景不一致，悬停反馈较弱。

**优化**：

- 增加统一背景容器（`width: 56px`、`height: 56px`、`border-radius: var(--radius-md)`）
- 使用 `var(--color-primary-soft)` 柔和背景 + `inset` 主色边框，暗色模式视觉一致
- 悬停时图标 `scale(1.1) rotate(-4deg)` + 主色光晕阴影，增强交互反馈
- 图标字号从 32px 调整为 28px（适配容器）

### 6. `src/pages/about.astro` - 增加页面 hero 区

**问题**：关于页直接使用 `<article class="container prose">`，缺少 `page-hero` 区，与全站其他页面视觉不一致。

**优化**：

- 外层包裹 `<section class="container">`
- 增加 `<div class="page-hero page-hero--tool">` 区，包含 h1 与描述段落
- 内层 `<article class="prose">` 保留文章式排版

### 7. `src/pages/blog/[...slug].astro` - 博客详情页标题装饰

**问题**：博客详情页 `.blog-article__content h2` / `h3` 缺少视觉装饰，章节识别度不足。

**优化**：

- `h2` 增加左侧主色渐变细条（`::before` 伪元素，3px 宽，`linear-gradient(180deg, primary, accent)`）
- `h2` 增加 `padding-left: 12px` 为细条留出空间
- `h3` 增加底部虚线装饰（`border-bottom: 1px dashed`），与 h2 形成层次差异

### 8. `src/pages/blog/[...slug].astro` - 补充图片与分割线样式

**问题**：博客详情页 Markdown 渲染缺少 `img` 与 `hr` 样式。

**优化**：

- 增加 `.blog-article__content img` 样式（圆角 + 边框 + 阴影 + 上下边距）
- 增加 `.blog-article__content hr` 样式（渐变透明分割线，柔化默认样式）
