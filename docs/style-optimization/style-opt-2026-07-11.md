## 具体修改内容

### 1. global.css - 语义色别名变量体系

新增一组语义化色值别名变量，作为工具页面统一引用的"中间层"，避免各页面反复硬编码 rgba 值：

```css
--color-danger: var(--color-error);
--color-danger-soft: var(--color-error-soft);
--color-danger-border: rgba(229, 72, 77, 0.3);
--color-warn: #d97706;
--color-warn-soft: rgba(245, 166, 35, 0.1);
--color-warn-border: rgba(245, 166, 35, 0.35);
--color-insert: #16a34a;
--color-insert-soft: rgba(22, 163, 74, 0.12);
--color-insert-border: #22c55e;
--color-delete: var(--color-error);
--color-delete-soft: rgba(229, 72, 77, 0.12);
--focus-ring: 0 0 0 3px var(--color-primary-ring);
```

暗色模式同步覆盖 `--color-warn: #fbbf24` 等变量，保证两种主题下语义色一致。

**设计意图**：将"危险/警告/新增/删除"四种语义从具体色值中解耦，后续调整配色只需改一处。

### 2. global.css - 工具页 hero 变体

新增 `.page-hero--tool` 变体样式：

- 标题下方增加主色渐变细线（`linear-gradient` 从主色到透明）
- 间距与字号针对工具页场景微调
- 与首页 hero 形成视觉层次区分，强化"工具页"专属感

### 3. global.css - 统一表单焦点环

新增工具页表单元素焦点统一规则，为 `.jsontool / .jwttool / .regextool / .hashtool / .qrtool / .pwtool / .colortool / .difftool` 容器内的 `input / textarea / select` 统一应用：

```css
box-shadow: var(--focus-ring);
outline: none;
```

**修复问题**：此前各工具页 focus 样式不统一，部分用 `outline: 2px solid`，部分用 `box-shadow`，视觉表现不一致。

### 4. global.css - 复制反馈动画增强

新增 `copy-pop` 缩放弹跳动画，与既有 `copy-flash` 组合：

- `copy-flash`：背景色短暂变绿（原有）
- `copy-pop`：按钮缩放 1 → 1.08 → 1 的弹跳反馈（新增）

组合后复制成功反馈更明显，提升用户感知。

## 优化前后对比

### 设计令牌体系

| 方面 | 优化前 | 优化后 |
|------|--------|--------|
| 语义色 | 各页面硬编码 rgba（如 `rgba(220, 38, 38, 0.08)`、`rgba(245, 166, 35, 0.12)`） | 统一通过 `--color-danger-soft` / `--color-warn-soft` 等语义变量引用 |
| 焦点环 | 各页面自定义 `outline: 2px solid` 或 `box-shadow`，表现不一 | 全部工具页统一 `box-shadow: var(--focus-ring)` |
| 字体引用 | 部分页面带冗余回退栈 `var(--font-mono, ui-monospace, SFMono-Regular, Menlo, monospace)` | 统一为 `var(--font-mono)` |
| 过渡曲线 | 大量硬编码 `0.15s ease` | 统一为 `var(--transition-fast)` / `var(--transition-base)` |

### 工具页 Hero

| 方面 | 优化前 | 优化后 |
|------|--------|--------|
| 视觉层次 | 与首页 hero 样式相同，无区分 | 新增 `.page-hero--tool` 变体，标题下方加主色渐变细线 |
| 工具页专属感 | 无 | 通过渐变细线强化工具页身份 |

### 复制反馈

| 方面 | 优化前 | 优化后 |
|------|--------|--------|
| 动画 | 仅 `copy-flash` 背景色变绿 | `copy-flash` + `copy-pop` 缩放弹跳组合 |
| 用户感知 | 较弱，易忽略 | 更明显，反馈更即时 |

### 维护性

| 方面 | 优化前 | 优化后 |
|------|--------|--------|
| 配色调整 | 需逐页面搜索替换硬编码 rgba | 只需修改 `global.css` 中语义变量 |
| 暗色模式适配 | 各页面单独覆盖硬编码值 | 暗色模式统一覆盖语义变量，各页面自动继承 |
| 新工具页开发 | 需重新定义一套色值 | 直接引用语义变量，自动符合全站风格 |
